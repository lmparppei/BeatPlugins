KEYBOARD SHORTCUTS

SCENE INFO

S Show/Hide Scene Numbers

P Show/Hide Page numbers in section headers

L Show/Hide Lengths inside scenes


SCRIPT NOTES & MARKERS

N Show/Hide Notes

  ⇥   Cycle through all notes one by one

M Show/Hide Markers


CHARACTER TRACKING

T Open Tracking window

1 through 9 Highlight scenes based on character presence (without going through Tracking window)

  ⇧   + number key adds character for highlighting.

0 Turn off any active character highlight


SELECTING SCENES

T Selects a scene

⌘A Select all (non-dimmed) scenes & sections

^A Select all (non-dimmed) scenes only


VIEWING MODES

F Toggle Flexible (proportional) mode

O Toggle One-column mode

X Index Card Modes: Cycle off, scenes, sequences


VIEW OPTIONS

⌘+ and ⌘- Increase and decrease font size

⌘0 Default font size

^⌘+ and ^⌘- Make Index Cards taller/less tall

C Toggle between Color Schemes


PRINTING

⌘P Print current outline


OTHER

R Toggle Real Time Updating

W Toggle Warnings for Delete/Move/Omit

⌘, Open Settings Window

? Open this screen

esc Exit any window, feature or FTOutliner (in that order)

⌘W Close FTOutliner
