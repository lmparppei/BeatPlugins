Manage Custom Entity Types

Custom entity types find entities using regular expressions (RegEx). Each pattern is run against every line of the script, scene by scene, and the text matched by the pattern's first capture group (…) becomes the entity's name. For example, PROP: ([A-Z ]+) turns every PROP: LANTERN in the script into a "LANTERN" entity. You can build and test patterns at regex101.com.
